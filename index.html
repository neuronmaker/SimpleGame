How to play:

Control the player with the keyboard. Steer it towards pellets to eat them. When there are no more pellets on screen, the game is over. Eat as many pellets as you can, eating pellets scores points and spawns new pellets. Get 'em before they're gone!

Start or unpause the game by pressing one of the movement keys.

Controls:

  • W → Up
  • A → Left
  • S → Down
  • D → Right
  • P → Pause the game
